I had a client of mine ask me to find a way to add their Twitter handle to the Twitter share text.

So I was able to find a work around but it's a bit of a hack until Squarespace releases better options for this feature. Basically, you will need to use some jQuery to inject a new Twitter button and suppress (hide) the old one with some CSS. It looks like the Twitter button that Squarespace uses is very basic and does not include any special parameters.
